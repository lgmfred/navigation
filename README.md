# Navigation

A Phoenix application as a navigation example.

Just defines three pages in `router.ex` (`"/home"`, `"/about"`, and `"/projects"`). Each page having its own controller as `HomeController`, `AboutController`, and `ProjectsController` and display a simple text response.

## Running the app

To start your Phoenix server:

  * Install dependencies with `mix deps.get`
  * Start Phoenix endpoint with `mix phx.server` or inside IEx with `iex -S mix phx.server`

Now you can visit [`localhost:4000`](http://localhost:4000) from your browser.
